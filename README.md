# 🎮 TK Client

Um aplicativo utilitário/modificador mobile para Minecraft PE desenvolvido nativamente em Android.

## 📋 Recursos

- 🎯 Menu Flutuante (Overlay) discreto
- 🔧 9 módulos de funcionalidades customizáveis
- 💾 Otimização de memória RAM
- 🎨 Interface premium minimalista

## 🚀 Como Usar

1. Clone o repositório
2. Abra no Android Studio
3. Clique em **Run** para compilar e instalar no dispositivo
4. Abra o app no seu phone
5. Clique no botão verde "INJETAR NO MINECRAFT"
6. O app fechará automaticamente após a injeção

## 📱 Funcionalidades

- ✅ Modo Zoom
- ✅ Nvision (Fullbright)
- ✅ Kill Aura
- ✅ Player ESP
- ✅ Player Cord
- ✅ Bridge
- ✅ Auto Totem
- ✅ CPVP (Crystal PVP)
- ✅ Aimbot
